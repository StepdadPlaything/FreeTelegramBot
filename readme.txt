It's Sample for east create telegram bot for smth. !all comments at .py file at Russian language.
Это бесплатный шаблон для создания телеграм бота для различных целей.
Среди функций:
1. Несколько комманд с отправкой ответных сообщений с вложениями или без.
2. Отправка ботом сообщения или голосового сообщения в ответ на разлицные сообщения от пользователя.
3. Рассылка всем пользователям, воспользовавшимся ботом.
Комманнда для рассылки - /broadcast "сообщение" ( без кавычек, комманду надо писать в чат телеграмма )
